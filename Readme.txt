Library Management System

System Features:
  1) Donate book
  2) Issue book
  3) Return book 
  4) Search book & book status

Entities & Attributes:
  1) Book(ISBN, Name, Author, Status)
  2) Student(Username, Borrow history, Current issued book)
  3) Admin(Username, Borrow history, Current issued book)
  4) Login Table(Username & Password)

Different Use cases (How Student/ Admin will interact with the system):
  Student Interaction Points
    1) Issue book
    2) Return book
    3) Search book
    
 Admin Interaction Points
    1) Issue book
    2) Return book
    3) Search book
    4) Donate book
